# tinybits
Pascal port of [tinybits](https://github.com/r-lyeh/tinybits).

- [x] Tiny bits and useful snippets.
- [x] Too simple to become libraries. Just cut & paste.
- [x] Cross-platform C/C++.
- [x] Public Domain.
